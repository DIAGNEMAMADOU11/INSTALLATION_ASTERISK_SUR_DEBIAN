# INSTALLATION_ASTERISK_SUR_DEBIAN
Projet académique présentant l’installation et la configuration d’un serveur de téléphonie IP (Asterisk) sur Debian Server, avec documentation détaillée et tests de validation.
# Installation et Configuration d’Asterisk sur Debian

## Objectif du projet
Ce projet documente l’installation, la configuration et les tests du serveur de téléphonie IP Asterisk sur Debian.

## Prérequis
- Debian 11 ou 12
- Accès root ou sudo
- Connexion Internet

## Structure du dépôt
- 01_installation : installation d’Asterisk
- 02_configuration : fichiers de configuration
- 03_tests : validation et tests
- 04_problemes_solutions : erreurs courantes et solutions
